# Kaulby

> Financial clarity for your business

**Live Site:** [https://kaulbyapp.com](https://kaulbyapp.com)

Kaulby helps small businesses and freelancers track expenses, manage budgets, and gain financial insights with beautiful, intuitive dashboards.

## Tech Stack

- Next.js 15 / React 19
- TypeScript
- Supabase
- Vercel
